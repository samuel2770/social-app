API Reference

Every Endpoint, Documented

All routes follow RESTful conventions. Protected routes require a Bearer token in the Authorization header.

POST /api/auth/signup

Register a new user. Returns a JWT token valid for 1 hour.

⬡ Public
POST /api/auth/signin

Authenticate with email & password. Returns a fresh JWT token.

⬡ Public
GET /api/auth/me

Returns the current authenticated user's profile with follower/following counts.

⬡ Protected
GET /api/posts

Paginated list of all published posts. Supports search, tag, sort (like_count, comment_count, timestamp).

⬡ Public
GET /api/posts/:id

Fetch a single published post with full author information embedded.

⬡ Public
POST /api/posts

Create a post. Always created as draft — publish separately when ready.

⬡ Protected
PATCH /api/posts/:id/publish

Changes a draft's state to published. Owner only.

⬡ Protected
PATCH /api/posts/:id

Edit title, content, or tags on a draft or published post. Owner only.

⬡ Protected
DELETE /api/posts/:id

Permanently delete a post. Cascades to remove all likes. Owner only.

⬡ Protected
GET /api/posts/me

Get your own posts. Filter by state=draft|published. Paginated.

⬡ Protected
GET /api/posts/feed

Personalized feed: published posts from people you follow, plus your own.

⬡ Protected
POST /api/posts/:id/like

Like a published post. Each user can only like a post once.

⬡ Protected
DELETE /api/posts/:id/like

Remove your like from a post. Returns the updated like count.

⬡ Protected
GET /api/users/:username

Public user profile with follower/following counts and is_following flag.

⬡ Public
POST /api/users/:id/follow

Follow another user. Cannot follow yourself or follow the same user twice.

⬡ Protected
DELETE /api/users/:id/follow

Unfollow a user you previously followed.

⬡ Protected
GET /api/users/me/following

Paginated list of users you are currently following.

⬡ Protected
GET /api/users/me/followers

Paginated list of users who follow you.

⬡ Protected

Tech Stack

Clean Architecture

Every layer is separated by concern. Easy to read, easy to extend.

🟢
Node.js + Express
Fast, non-blocking server with clean MVC routing.
🍃
MongoDB + Mongoose
Flexible document storage with powerful query indexing.
🔐
JWT Authentication
Stateless auth with 1-hour token expiry and optional middleware.
🧪
Jest + Supertest
Full integration test suite with an in-memory MongoDB database.
⚡
In-Memory Test DB
Tests run in isolation using mongodb-memory-server.
📦
Clean MVC Pattern
Models → Controllers → Routes — each file does one thing well.
